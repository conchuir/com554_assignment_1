Vim

The Programmer's Text Editor

Vim is probably the most basic text editor you'll find in use today. It's highly configrable and comes installed on most UNIX and MacOS systems. Vim is highly regarded and well used by developers because of it's efficiency and configurability.

What is Vim?

Vim is Vi Improved. Vi is an old plain text editor with minimal features made in 1976. In 1981, a software genius decided that Vi was no longer enough to satisfy developer and sysadmin needs, and so developed an improved version of Vi, named Vi Improved, or Vim for short.
Vim is a very popular text editor due to its immense customization options and high performance. Vim also works over SSH connections, so you can create and edit remote files just as you would local ones. It has support for hundreds of development languages and creates small files that keep app sizes small.
It's so popular in fact, that in 2006, Linux Journal readers voted it the most used text editor.
